\noindent\textbf{Résumé :}

\noindent Ce projet présente la conception et le développement d'un microservice DevSecOps de type Policy-as-Code, dédié à l'analyse automatisée de la sécurité du code d'infrastructure avant le déploiement. Le service analyse le code Terraform, Dockerfile et Kubernetes YAML, applique 100 politiques de sécurité couvrant 14 catégories, et rend des décisions ALLOW ou BLOCK en 46 millisecondes en moyenne. L'architecture à 4 couches intègre une couche de normalisation innovante qui réduit la complexité du code de 67\%. Le service s'intègre aux pipelines CI/CD Jenkins et a été démontré à l'aide d'une simulation hors ligne complète utilisant Gitea comme serveur Git local.

\vspace{1cm}

\noindent\textbf{Abstract:}

\noindent This project presents the design and development of a DevSecOps Policy-as-Code microservice dedicated to the automated security analysis of infrastructure code prior to deployment. The service scans Terraform, Dockerfile, and Kubernetes YAML code, enforces 100 security policies across 14 categories, and renders ALLOW or BLOCK decisions in approximately 46 milliseconds on average. The 4-layer architecture integrates an innovative normalizer layer that reduces code complexity by 67\%. The service integrates with Jenkins CI/CD pipelines and was demonstrated using a complete offline simulation featuring Gitea as a local Git server.

\vspace{0.3cm}
\noindent\textbf{Keywords:} DevSecOps, Policy-as-Code, Infrastructure Security, Static Analysis, FastAPI, Terraform, Docker, Kubernetes, CI/CD, Jenkins, Shift-Left Security, Microservice.
